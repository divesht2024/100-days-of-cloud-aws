# ☁️ 100 Days of Cloud (AWS) Challenge

Welcome to my **100 Days of Cloud** journey! 🚀

This repository documents my daily progress as I learn and practice **Amazon Web Services (AWS)**, **Cloud Computing**, **DevOps**, and **Infrastructure Automation** through hands-on labs, projects, and real-world scenarios.

## 🎯 Goal

The goal of this challenge is to build a strong foundation in Cloud and DevOps by consistently learning, practicing, and documenting my progress for **100 consecutive days**.

## 📅 Challenge Rules

- 📚 Learn something new every day.
- 🛠️ Complete hands-on labs and projects.
- 📝 Document daily learnings.
- 💻 Push updates to GitHub regularly.
- 🚀 Focus on practical implementation rather than theory alone.
